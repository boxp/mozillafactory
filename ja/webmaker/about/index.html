Web で何かすごいものを作ろう

Mozilla Webmaker は、あなたが Web で何かを作ることをお手伝いします。私たちには、新しい ツール と、始めるための プロジェクト、グローバルなクリエーターたちの コミュニティ があります。このコミュニティには、教育者、映画制作者、ジャーナリスト、開発者、若者などが参加し、すべての人が共に作り、学びます。

目標: 数百万人の人々が「Web を使う」ことから「Web を作る」ことへ前進する手助けをすることです。Mozilla の非営利の ミッション の一部として、世界中の人々が Web を理解し、彼らのオンライン上での活動を助け、Web を読み書きできる人々を育成したいと考えています。

Web メーカー世代の育成

Mozilla Webmaker が提供する具体的なもの:

  1. ツール：私たちのコミュニティが設計し、構築したオーサリングツールとソフトウェアです。Popcorn で Web 動画にさらに力を与え、X-Ray ゴーグル で Web ページを調べてリミックスし、Thimble で自分だけの Web ページを作ります。Web メーキングのためのスイス・アーミーナイフや「スーパーヒーローの変身ベルト」のようなものです。
  2. プロジェクト：すべての人々が Web で何かすばらしいモノを作ることを助ける、実用的なスタータープロジェクトとその使い方、レシピです。ブログのテンプレートを書き換えることから、世界を変えるアプリを構築することまで。
  3. コミュニティ：様々なスキルと経験を持つ人々が集まります。教師、映画制作者、ジャーナリスト、若者など。Web に詳しい人から初心者まで、すべての人々が様々な場所でイベントや集会、ハックジャムを開催し、そこでモノを作ったり学んだりします。

なぜ Mozilla が取り組むのですか？

Web は、21 世紀における重要なスキルであり、世界の第 2 言語になりつつあります。今日のデジタルリテラシーは、読み書き計算と同様の重要な能力です。Mozilla は、人々に理解する必要のあるスキルを与え、このような世界をただ消費するのではなく、形作り、積極的に関与していくのは、非常に重要なことだと信じています。

“Web メーカー” とは何ですか？

オープンな Web を構築するブロックを使って何かをする人です。 あなたが写真や動画をアップロードしたりブログを投稿したり、ソーシャルメディアを使っているなら、あなたはすでに Web メーカーです。私たちは、あなたのスキルアップを助け、さらにその先へ進む手助けをしたいと考えています。

Web メーカーになるには、プロのコーダーや達人になる必要はありません。 パンを焼くのにパン職人になる必要は無く、タイヤを交換したりボンネットを開けたりするのに専門の自動車工になる必要が無いのと同じように、Web の仕組みを理解したり Web ページを書いたりするのにプロの Web 開発者になる必要はありません。私たちは、「アップロードの先にある」あなたの人生と仕事における創造と革新のための新たな機会を与え、あなたが先へ進むことを助けたいのです。

Web の未来を形作ろう、共に

私たちは、共に、Web を変えるテクノロジーを作っています。 Mozilla Webmaker プロジェクトの Popcorn は、Web 動画の外面を変えます。 Thimble は、初心者向けに Web が書かれている言語の謎を解き明かします。Hackasaurus は、デジタルリテラシーと学習に力を与えます。 OpenBadges は、あなたの可能性を解き放ち、そのスキルを認めます。 OpenNews は、ニュース編集室における Web 革新の触媒になります。 Hive NYC は、若者にテクノロジーの未来をデザインする能力を与えます。

これが Web メーカーのすべてです: 様々な人を集めてダイナミックな新しい Web プロジェクトとテクノロジー、アプローチ、学習を創造します。私たちは、Web の開かれた可能性に情熱を感じ、人々に Web の未来をデザインして構築する力を与えたいと願っています。

私も参加できますか？

はい！ Mozilla Webmaker では、あなたのためのツールとプロジェクトをご用意しています:

教育者
デジタルリテラシー、HTML と CSS のような Web 制作の基本を教えるためのリソース、あなたが主催する学習イベントのための ガイド があります。
映画制作者
Mozilla Popcorn は、映画制作者やドキュメンタリーの作家、マルチメディアアーティストが彼らのオンライン動画をポップにします。
ジャーナリスト
Knight-Mozilla OpenNews は、ニュースハッカーの仲間とソフトウェア、知識を提供します。
若者
Hive NYC は、若者のためのデジタル学習ラボです。Hackasaurus は、若者が Web ページを調べてリミックスすることを助けます。
開発者とデザイナー
Mozilla のツールとプロジェクト に貢献することから、世界を変えるあなた独自の 次世代アプリ をデザインして構築することまで。
初心者
私たちの プロジェクト は、易しいものから難しいものまで分類され、Web メーキングを簡単に始める手助けをします。
上級者
今まで誰も見たことの無いような実験的なアプリ、デモ、Web エクスペリエンスを創造します。

参加しよう

Mozilla Webmaker ツールとプロジェクトは、あなたのようなクリエイティブな人々が参加するハンズオンを通して、オープンにデザインされ、構築され、テストされました。あなたも以下のことに参加できます:

  • Webmaker のコミュニティコールに参加しよう。 どなたでも歓迎です。毎週火曜日にミーティングをします。スペシャルなゲストスピーカーがひょっこり現れるかもしれません。現在の状況を知り、あなたのアイデアを共有してください。
  • Summer Code Party に参加しましょう。 私たちは、世界中で Webmaker ツールとプロジェクトを使って作り、学ぶイベントをこの夏に開催します。 お近くの イベントを見つける か、イベントを主催 してください。
  • 最新情報を受け取ろう。 メーリングリストを購読して最新の Webmaker ニュースと進行状況を受け取ってください。
  • コンタクトしよう。 Webmaker コミュニケーションチャンネルのリストと連絡方法のリストです。Webmaker ニュースグループから公式のチャットルーム、特定のプログラムまで。
  • よくある質問を読もう。 そして、あなたの質問を加えてください。
  • あなたの成果を共有しよう。Webmaker で何かすごいものが出来ましたか？ ぜひ、私たちに教えてください。#webmaker ハッシュタグか Summer Code Party の #mozparty ハッシュタグを付けてツイートしてください。
  • あなたの作った Webmaker プロジェクトや How-To を投稿しよう。 何かすごいものを作ったり学んだりしましたか？ 他の人もそれができるように見せてください。あなた独自の一歩ずつ進める How-To を作って、それを Webmaker プロジェクトギャラリーに 投稿してください。
  • イベントに参加またはイベントを主催しよう。 便利な イベントガイド を読んで、大規模なハックジャムからキッチンテーブルに集まる小規模なイベントまで、イベントを開催してください。
  • Mozilla に参加しよう。 Mozilla の非営利のミッションを支援するための寄付をお願いします。(誰もがほしがる Firefox T-シャツを差し上げます。この方法でしか入手できません！)
