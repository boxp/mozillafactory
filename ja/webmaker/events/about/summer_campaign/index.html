Mozilla Summer Code Party に参加しよう

私たちは Webmaker (ウェブメーカー) の世代を育成しています。最初のステップとして、様々なところから人々を呼び集めて「出会い」の場を設け、何かステキなものを「作り」、Web の背後にあるコードを「学び」ます。私たちは、この Summer Code Party (サマーコードパーティ) と呼ばれるイベントを 6 月 23 日に開始します。このイベントは、Mozilla と他の多くのパートナー組織が共同で開催します。

参加するには:

  1. 1 友人を誘ってイベントに行きましょう。
  2. 2Web 上で何かステキなモノを作りましょう。
  3. 3Web がどのように作られているかを学びましょう。

イベントの概要

何を？

人々が出会い、Web のために何かを作る方法を学ぶイベントです。

いつ？

6 月 23 日から 9 月 23 日までです。

どこで？

どこでも開かれます。ご自宅のキッチンテーブル、オフィス、近所の図書館など。

なぜ？

Web がどのように作られているか、コードがどのようにを書かれているかをすべての人に知ってもらうためです。

どのように？

出会いの場を設けます。教えるためのボランティアを募ります。またはイベントパートナーになりましょう。

日本での開催情報

Mozilla Summer Code Party in Kyoto では、Thimble (シンブル)、Hackasaurus (ハッカザウルス)、Popcorn (ポップコーン) といったツールを使用して、楽しみながら“Web の読み書き”を学ぶ予定です。

当日は、Mozilla Japan のスタッフや大学院生たちがチューターとなり、サポートします。リラックスした雰囲気の中、ジュースやお菓子を食べながら、ゲーム感覚で楽しみながら Web の仕組みを学びましょう。

主な対象者を、中学生 ～ 大学生としていますが、もちろん、小学生から大人まで、ご興味のある方は、ぜひご参加ください (6 月に東京で開催した時は、小学校低学年から 80 才近い方まで参加されました)。Web に興味があれば、技術的な予備知識はなくても構いません。ぜひ、お越しください！

参加を希望される方は、事前に フォーム からお名前・年齢・来場予定の時間 (目安) をお知らせください。来場者人数把握のため、ご協力お願いいたします。

よくある質問

Mozilla の Summer Code Party とは何ですか？

Mozilla が主催するサマーキャンペーンです。6 月 23 日から始まり、夏の間中、世界各地で開催されます。一緒に Web でモノづくりをすることにより、Web のコードを書くための基礎を学びます。ツールと教材を使って誰でも学べる、100% 自由でオープンな楽しいイベントです。

誰が参加しますか？

Web を学びたい方、ツールを使ってコードの書き方を学びたい方はどなたでも参加できます。または、一般的な Web についての知識を増やすのもよいでしょう。

  • ボランティアの教師とインストラクター。コードの匠や詳しい人から、キッチンテーブルの周りで学ぶ人まで。
  • 開発者、デザイナー、オープン Web おたく。Web の学習ツール、リソース、カリキュラムの構成を助けます。
  • パートナー組織とコミュニティ。私たちの大きな「日傘」の下に集まり、一緒に作り、学びます。

Mozilla は何を用意していますか？

  • ツール：コードに対する恐れを取り除くソフトウェア、オーサリングツール、Web 制作アプリを提供します。これらのツールで何かを作ることにより、人々が学ぶことを助けます。
  • 教材：レシピ、キット、カリキュラムと使い方の資料を提供します。楽しい夏のプロジェクトを誰でもすぐに使えます。Web は、あなたが最初に鳥小屋を作るのに似ています。または万能包丁のようなものです。
  • イベント：催し物、キャンプ、学びの場、キッチンテーブルで世界中の人々が出会います。さらに、あなたのイベントを共有する手段とプラットフォームを提供します。
