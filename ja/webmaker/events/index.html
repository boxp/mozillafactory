Mozilla は、この夏皆さんが Web 上でステキなものを作るお手伝いをしたいと考えています。例えば、簡単なコードの書き方を学んでみませんか。世界各地で行われる Summer Code Party イベントに参加するか、ボランティアとして教える側に回ってください。すぐに使えるツール、モノづくりプロジェクト、そして楽しいイベントを用意しています。イベントの主催者も募集しています。
