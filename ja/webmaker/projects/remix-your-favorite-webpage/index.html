お気に入りの Web ページをリミックスしよう

不思議な X-Ray ゴーグルを使い、Web ページの裏側にあるコードを探索して書き換えてみよう。

X-Ray ゴーグルを使うと、Web ページを作り上げる要素を覗いてリミックスできます。ゴーグルを装着して Web が何でできているか調べ、テキストや画像を魔法のように置き換えてください。

はじめよう

  1. X-Ray ゴーグルをインストールしてください。 ゴーグルは、ブラウザで動作する単純な「ブックマークレット」です。インストールは簡単です。この手順に従ってください。
  2. お気に入りの Web サイトを開いてください。 ゴーグルはどこでも使えます。
  3. X-Ray ゴーグルを装着してください。 ブラウザのブックマークツールバーにインストールした X-Ray ゴーグルのボタンをクリックするだけです。

    この動画でゴーグルの使い方を紹介します。


  4. Web ページを探索しましょう。 マウスを動かしてページ内のいろいろなところを見てください。ゴーグルがページの裏側にある HTML と CSS 要素を表示します。
  5. ページをリミックスしましょう。 リミックスしたい要素の上にマウスを動かしてキーボードの 'R' キーを押してください (心配しないでください――あなたのコンピュータにコピーされたデータを書き換えるだけで、他の人が見る実際のページを書き換えるわけではありません)。ページのテキストを書き換えてみてください。画像を置き換えるのもよいでしょう。例えば、Google のトップページの画像をペットの画像に置き換えてみましょう。
  6. 画像を置き換えるには: はじめに、置きたい画像の URL が必要になります。まだ画像を決めていない場合は、Google 画像検索 で何か検索してみましょう。次に、画像を右クリック (Mac の場合は Ctrl+クリック) して「画像の URL をコピー」を選択します。あとは、ゴーグルを使って画像の URL の部分に貼り付けるだけです。
  7. 要素のスタイルを変えましょう。 テキストの色やフォントなど、要素のスタイルを編集するには、キーボードの 'C' キーを押しながらスペースキーを押してください。
  8. リミックスしたページを共有しましょう。 リミックスしたページの スクリーンショット を撮って、#mozparty のハッシュタグを付け、友だちと共有してください。
