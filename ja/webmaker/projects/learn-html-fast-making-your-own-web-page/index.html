独自の Web ページを作って HTML を学ぼう

Mozilla の Thimble アプリを使ってあなたの最初の Web ページを作り、HTML の基本を学びます。

このスタータープロジェクトは、シンプルな HTML エディタを使い、最も一般的な HTML 要素とタグを書いてみます。

これは、他の Webmaker プロジェクトに必要な基本スキルを向上させるとても良い手段になります。または、あなたのプロジェクトを始めるのにも良いでしょう。まだ HTML を書いた経験が無くても、このプロジェクトが最初の一歩になります。とても簡単に始められることに驚くでしょう。
