ポップアップビデオを作ろう

Web 動画にポップアップとリンクを追加しましょう。

Mozilla Popcorn のインタラクティブな動画ツールを使って、コメント付きのテキストボックスを追加し、アクションを呼び出し、Web 動画上にリンクを追加してください。

はじめよう

  1. 動画を選びましょう。 あなたの制作した動画を使うか、既存のものから選んでください。YouTube や HTML5 の動画が使えます (例えば、Prelinger Archive には数多くの流用可能な古い CM の動画があります。ページ下部から "ogg" 形式のものを選んでください)。
  2. テンプレートを開きます。 Mozilla Popcorn の Pop-up テンプレート を開いてください。
  3. 動画を追加しましょう。 動画左上の "Edit Source" ボタンをクリックして、選んだ動画の URL をテキストボックスに貼り付けてください。
  4. ポップアップテキストを変更しましょう。 タイムライン上の "POP" イベントをダブルクリックして、"Text" フィールドのテキストを変更してください。別のページや動画へのポップアップリンクを作るには、"Link" フィールドに URL を貼り付けます。
  5. ポップアップの配置とスタイルを変えましょう。 "POP" イベントをダブルクリックし、左側の "Position" タブを選択します。スライダを動かして好きな位置にポップアップを移動してください。"Time" タブでは、ポップアップが現れる時間とフェードアウトの時間を変更できます。編集が完了したら、画面右下の "Done" ボタンをクリックしてください。
  6. ポップアップを追加しましょう。 画面右下の "My Events" から pop をタイムライン上にドラッグするだけで追加できます。
  7. あなたの作品を公開して保存しましょう。 すべての編集が完了したら、画面右上の "Publish" ボタンをクリックしてください。Mozilla の "Persona" サービス (BrowserID) を利用してログインするウィンドウが開きます。ログインしたら、動画の名前を入力して "Save" をクリックします。この動画を見るための URL が表示されます。この URL をコピー＆ペーストして共有してください。
  8. 作品を世界中の人と共有しましょう！ あなたの作品の URL を、メールや Twitter などで共有してください。右上の"View Source" ボタンをクリックすれば、あなたの Web ページに貼り付けられる HTML のコードが表示されます。
感想をお寄せください。 このプロジェクトを改善するため、フィードバック をお寄せください。
