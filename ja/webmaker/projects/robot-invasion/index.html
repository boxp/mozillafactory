ロボット侵略ビデオを作ろう

ロボットたちが町を侵略しようとしています！

Popcorn インタラクティブ動画ツールを使って、テキストやロボットの話す言葉、地図、写真を変更しましょう。

はじめよう

  1. テンプレートを開きます。 Mozilla Popcorn の Robot Invasion テンプレート を開いてください。
  2. 動画を選びましょう。 テンプレートに読み込まれている "Robots Invade Everytown" の動画を使うことができます。別の動画を使いたい場合は、動画左上の "Edit Source" ボタンをクリックして、選んだ動画の URL をテキストボックスに貼り付けてください。YouTube や HTML5 の動画が使えます
  3. タイトルを書き換えましょう。 タイムライン上の "TITLES" イベントをダブルクリックしてタイトルを書き換えてください。完了したら右下の "Done" ボタンをクリックします。
  4. ロボットの話す言葉を変えましょう。 とても簡単です。タイムライン上の "SPEAK" イベントをダブルクリックして、"Text to speak" のテキストボックスの言葉を書き換えてください (英語のみ)。あなたの入力した言葉がロボットの声で話されます。魔法のようでしょう！
  5. 地図を変えましょう。 タイムライン上の "GOOGLEMAP" イベントをダブルクリックして、地図上の場所を好きなところに変えてください。通りの所在地を入力するか、新しい場所の緯度と経度を入力してください。または、画面左下の地図をドラッグしてください。"Map Mode" のドロップダウンリストから地図のモードを変更することもできます。
  6. 画像を追加しましょう。 タイムライン上の "PHOTO" をダブルクリックして、"Source URL" フィールドにあなたの写真の URL を貼り付けてください。(上級者向けのヒント: Google 画像検索で好きな画像を検索してください。画像を右クリックして "画像の URL をコピー" を選択し、その画像の URL を貼り付けてください。)
  7. "Zoinks" を追加しましょう。 "Zoinks" は、動画上に現れるテキストのポップアップです。"ZOINK" イベントをダブルクリックしてテキストを変更してください。ドロップダウンメニューからスタイルや配置など、他のオプションも変更できます。
  8. イベントを追加しましょう。 イベントを追加したり編集したりしてあなただけの動画を作りましょう。画面右下の "My Events" からイベントをタイムライン上にドラッグしてイベントを追加できます。
  9. あなたの作品を公開して保存しましょう。 すべての編集が完了したら、画面右上の "Publish" ボタンをクリックしてください。Mozilla の "Persona" サービス (BrowserID) を利用してログインするウィンドウが開きます。ログインしたら、動画の名前を入力して "Save" をクリックします。この動画を見るための URL が表示されます。この URL をコピー＆ペーストして共有してください。
  10. 作品を世界中の人と共有しましょう！ あなたの作品の URL を、メールや Twitter などで共有してください。皆に見てもらうために、Twitter で #AllHailRobots タグを付けてあなたの動画をツイートしてください。ロボット万歳！
感想をお寄せください。 このプロジェクトを改善するため、フィードバック をお寄せください。
