「バニー・マッシャー」で遊ぼう

HTML と CSS を使ってバニー君を安全な場所に移動します。

かわいい小さなバニー君をいじめたいのは誰？ バニー・マッシャーがバニー君に襲いかかろうとしています。あなたの HTML と CSS のスキルを使ってバニー君を危険な場所から安全な場所へ移動してください。さもなくば・・・あなた次第です。あなたのクリエイティブなハックで場面を書き換えましょう。
