Web 上に世界地図を作ろう

お気に入りの Web サイトや画像、場所が出てくる、昔のビデオゲームのような地図を作りましょう。

HTML を使って、昔のビデオゲームに出てくる地図をリミックスし、Web 上のアイテムを使った地図を作ります。画像やテキスト、場所へのリンクを追加して、完成した地図を友人と共有し、あなたの作った地図を探検してもらいましょう。
