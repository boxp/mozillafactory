Thimble

Webmaking made easy

Mozilla の Thimble アプリで、びっくりするほど簡単に、独自の Web ページを作って共有することができます。ブラウザ内で HTML と CSS を書いて編集してください。その場で、書いたものがプレビューできます。出来上がった作品は、クリック 1 つで共有できます。

使ってみよう

Thimble の活用事例

  • 独自の Web ページを作って HTML を学ぼう
  • Web 上に世界地図を作ろう
  • 「バニー・マッシャー」で遊ぼう
  • Webmaker アバターを作ろう
  • 「ゾンビを追い出せ」で遊ぼう
  • 他の Thimble プロジェクト (英語)

X-Ray ゴーグル

Web ページの仕組みを調べよう

X-Ray ゴーグルで、Web を構成している様々なブロックを簡単に調べることができます。ゴーグルを装着して、New York Times のサイトからあなたのブログまで、Web ページの裏側にあるコードを調査してみましょう。調べたコードのテキストや画像を置き換えて、クリック 1 つでリミックスしましょう。

使ってみよう

X-Ray ゴーグルの活用事例

  • お気に入りの Web ページをリミックスしよう
  • 他の X-Ray ゴーグルプロジェクト (英語)

Popcorn

Web 動画にエネルギー充填

Mozilla Popcorn は、退屈なオンライン動画をダイナミックな「ハイパービデオ」に変え、動画に豊かな表現とコンテキストを与えます。Popcorn を使って、再生中の動画に別の Web サイトへのリンク、ソーシャルメディアやニュースフィード、データの視覚化、他のコンテンツをリアルタイムで追加できます。ポップアップ動画やマルチメディアレポート、ガイド付きの Web ツアーなど、独自の動画を作ってみましょう。

使ってみよう

Popcorn の活用事例

  • ポップアップビデオを作ろう
  • ロボット侵略ビデオを作ろう
  • 他の X-Ray ゴーグルプロジェクト (英語)
