X-Ray ゴーグルのインストール

申し訳ありませんが、ゴーグルはまだ Microsoft Internet Explorer で動作しません。

最新バージョンの Firefox または Opera、Chrome、Safari のいずれかをご使用ください。

X-Ray ゴーグルは ブックマークレット ―― Web ページに機能を補う特殊なツールです。

このページでは、ゴーグルのブックマークレットをインストールする方法と Web 上のものを調べてリミックスする方法を説明します。

  1. この動画を見てブラウザのブックマークツールバーを表示してください:

    タブバーの空いたところを右クリックし、[ブックマークツールバー] を選択してチェックを入れます。

  2. この X-Ray ゴーグル をブックマークツールバーにドラッグしてください。

  3. 開いた Web ページをリミックスするには、ブックマークツールバーの X-Ray ゴーグルボタンをクリックするだけです。

これで終わりです。Web メーキングを楽しんでください！
