  • Summer Code Party in Kyoto

    私たちは Webmaker (ウェブメーカー) の世代を育成しています。最初のステップとして、様々なところから人々を呼び集めて「出会い」の場を設け、何かステキなものを「作り」、Web の背後にあるコードを「学び」ます。私たちは、この Summer Code Party (サマーコードパーティ) と呼ばれるイベントを 6 月 23 日に開始します。このイベントは、Mozilla と他の多くのパートナー組織が共同で開催します。

    開催日時
    7月25日（水）11:00 - 17:00
    会場
    京都リサーチパーク町家スタジオ
    対象者
    主に中学生〜大学生
    主催
    一般社団法人 Mozilla Japan
    協力
    京都クリエイティブワークショップ
