プロジェクト

Mozilla Factory では、プロジェクトごとにメンバーを募集し、活動します。

プロジェクトは、自分たちがやりたいと思うテーマをプロジェクト化する方法と、外部機関 (研究、教育機関、企業など) から提供されたテーマなどをもとに、プロジェクト化する方法の 2 通りがあります。

プロジェクトを通して、“考える”、“創造する”、“作る”、“伝える”、“教える”の要素を体験的に学べるだけでなく、それに関わる様々なジャンルの人々と、コミュニケーションを図ることもできます。

また、プロジェクトのプロセス、参加メンバーや進捗状況は、いつも Web 上で“オープン”に公開されることを原則としているので、プロジェクトの状況をだれでも把握することができます。

ここではオープンにできないプロジェクトに関しては行うことができません。

Mozilla Factory で、どんな“モノ”が生まれるのか、ぜひご期待ください！

現在活動中のプロジェクト

Webmaker

Web は、21 世紀における重要なスキルであり、世界の第 2 言語になりつつあります。Web の仕組みを理解したり Web ページを書いたりするのにプロの Web 開発者になる必要はありません。Web の読み書きを覚えて、Web で何かすごいものを作ろう。

Mecha-Mozilla

「Web」×「実世界 (リアル)」のいい感じの関係性を考えながら、Web の可能性を追求するモノづくりプロジェクト。現在、味と香りで Web サイトを楽しむ「weburette」など 3 つのサブプロジェクトを展開中。

プロジェクト募集

こんなプロジェクトがしたい、前からこんなことをやってみたかった、こんなことができたらいいな、などプロジェクトの題材となるものがあれば、ぜひ Mozilla Factory にプロジェクトテーマを提供してみませんか？

ただし、プロジェクトテーマ提供者のためだけに活動するわけではなく、オープンプロジェクトとして自分のため、世の中のために活動します。

ここでの成果物は、オープンソースの概念の下、皆に共有されます。
