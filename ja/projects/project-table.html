現在活動中のプロジェクト

ぶっとびケータイ

どんなケータイがあったら嬉しいですか？見ているだけでワクワクする、使うと 仲良し家族になる、ポケットに入れるとヒンヤリ気持ちいい、恋人の気分によっ てカタチが変わる。「ぶっとびケータイ プロジェクト」では、みんなでアイデ アを出し、コンセプトを作り、デザインを考えて、試作品を作るところまでを目 指して活動する、ぶっとんだメンバーを募集中です！

Mecha-Mozilla

「Web」×「実世界 (リアル)」のいい感じの関係性を考えながら、Web の可能性を追求するモノづくりプロジェクト。現在、味と香りで Web サイトを楽しむ「weburette」など 3 つのサブプロジェクトを展開中。

Webmaker

Web は、21 世紀における重要なスキルであり、世界の第 2 言語になりつつあります。Web の仕組みを理解したり Web ページを書いたりするのにプロの Web 開発者になる必要はありません。Web の読み書きを覚えて、Web で何かすごいものを作ろう。

プロジェクト募集

こんなプロジェクトがしたい、前からこんなことをやってみたかった、こんなことができたらいいな、などプロジェクトの題材となるものがあれば、ぜひ Mozilla Factory にプロジェクトテーマを提供してみませんか？

ただし、プロジェクトテーマ提供者のためだけに活動するわけではなく、オープンプロジェクトとして自分のため、世の中のために活動します。

ここでの成果物は、オープンソースの概念の下、皆に共有されます。
